Youtube embedding!

I have now configured a youtube tag to be used in BBCode!

How do you use it?
Get the url of the video you want to embed (Youtube only works)

Code:
http://www.youtube.com/watch?v=YuOBzWF0Aws
Get the last text after ?v= on the url.
Code:
YuOBzWF0Aws

Then add the youtube tag like so
Code:
[youtube]YuOBzWF0Aws[/youtube]



I hope you enjoy!
